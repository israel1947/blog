# Blog App

## Instalación

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Backend
1. `cd backend`
2. `npm install`
3. `node server.js`

### Ejecutar ambos proyectos
Desde el directorio raíz, puedes ejecutar el frontend y el backend juntos:
1. Modo Dev: `npm run start:all:dev`
3. Modo prod: `npm run start:all:prod`
